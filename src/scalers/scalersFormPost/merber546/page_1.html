1.     完成ScalersTalk成长会coding小组的进群任务（CodeCademy python部分）。

2.     在ScalersTalk成长会coding小组实际学习的三个月期间学习《python核心编程 第二版》到第10章，在bing、Baidu、S开头的外国问答网站（叫啥来着？）等等的帮助下完成课后习题。

3.     在Amazon购买了科技、编程相关书籍阅读。

4.     考完试后在购买了crossin先生的基础python课程学习。

感想与总结：

1.     英语能力的缺乏。在CodeCademy学习期间，卡进度的原因之一就是后期教程的理解（经常连提问的要求看不懂）。解决方法：查字典，加上网友提供的代码揣摩。（然而国外友人的英语我也看得头疼无比）

2.     作为python新手在群里认为群友没有义务解答我学习途中的疑惑而经常不好意思问看起来很傻的问题。为了跟上群里复盘进度，途中会不由自主地刷进度。囫囵吞枣的结果也就是半桶水啦，到后期课后练习也很难解决，概念性的问题也迷迷糊糊的。期末考试前请了一个月假期复习，过后就对跟上进度无望，以主动退群告终。

在进群学习python之前，只是想当然地认为，编程嘛，不就是学习语法编写一段程序实现某种功能。现在才了解跟本就不是那回事啊；_；

自我反思，没有持续的原因如下：

o   没有及时反思总结学习途中的问题，有意无意地片面追求GDP高速发展，殊不知国民生活水平的提高由多方面因素影响。

o   没有将其应用于实际情况，减少重复劳动工作，给个人生活提供便利。只是单纯地输入没有输出，印象不够深刻。

3.     书籍包括：

o   失控（Out ofControll)（4/24）

o   必然（TheInevitable) by Kevin Kelly

o   信息简史（TheInformation:a History,a Theory,a Flood）by James Gleick（15/15，缺少背景相关知识，一知半解）

o   Python数据分析与挖掘实战 by 张良均（基础篇通读完，后续因为相关数学知识的缺乏无法深入）

在订阅了李笑来老师的公众号（阅读信息简史后，失控前），了解到十分钟栏目后，尝试在阅读时以此为模板写略写，概括章节大意。虽然信息简史、失控都不是浅薄的书，但以此行动后感觉理解的程度还是有一定差别的。目前失控只阅读到第四章节（为了行文流畅与言简意赅经常需要反复阅读以致进展缓慢），继续行动。

数据分析与挖掘实战阅读途中纠结于各种概念……还是没有相关数据分析基础的学习，在与其他学科结合后阅读就显得吃力。

4.     找对了自己的定位，基础学习者-_-|||可能之前在群太紧张，现在很轻松，在群里经常会问些小白问题（捂脸）。感觉crossin先生做的学习导航还是挺友善的，也有附录一些实用或有趣的小例子。希望在学习完后可以入门WWW

P.S：期间花了自己搭好了XX-NET，开心了好久

在语言方面

英语

1.     进入听力零基础群练习听力（70p→85p)。

2.     GRE考试（141+156+1）

日语

1.     开启了日语学习（初上）。

2.     （异常艰难地）啃了本生肉。

感想与总结：

1.     这个学期的英语学习懈怠了。虽然一直有按频率听着VOA，但都没有使着力，也没有及时复习总结错词。书面练习也少了，还是不习惯英文阅读（日常阅读中看见英文会下意识略过）。初步设想的解决方法是，硬着头皮杆，被动输入与主动输入结合。计划没有完美的，先上路再完善。争取每天都有时间练习。

2.     没有平衡好GRE备考与python学习。多线程学习计划崩了。单词没有反复练。

3.     接触日本文化许久（主要来源动漫），却是最近才开始学习这门语言。初学词汇都是苦手，不过还好有汉字可以辅助记忆，然而面对一堆单纯的平假名\片假名组成的单词或句子我常常是懵逼的，往往得读出来才知道是啥意思。

4.     有需求才有动力啊！明明看课本时要死要活，但为了读本可以硬生生坐一个下午拆结构划单词，bingMoji小D齐上阵,老师被我问烦了干脆直接要了图源帮我全部翻了_(:3 J L)_

在运动方面

1.     坚持三个月每天400*10m

2.     学习了一个月的钢管舞

感想与总结：

1.     跑完步后洗澡特别舒服，心情也变好了。但两个月之后膝盖开始出现疼痛的情况，于是停止。感觉单一的运动还是有点枯燥，以后可以试试游泳、徒步等结合。

2.     累……运动量大，一天课下来往往是条废狗的状态。个人皮肤较薄，初学一两个星期，两条腿都是花的。然而平时注意饮食的话，是个减肥的好方法。

可怕的是

这个学期竟然挂科了……这是平时不听课，经常逃课，期末又不专心复习的后果啊。
