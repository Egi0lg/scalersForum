1.你过去的2015是什么样子的？请用三个关键词来描述一下。围绕每个关键词你可以讲一两个故事，或者对你自己的表现进行评价，包括你的行动状态，心理情绪，身体状态等各方面的稳定性以及完成了什么样的事情。

 

焦虑，挣扎，自我接纳

2015年2-8月这段时间在一家很小的外贸服装公司上班，每天上下班占用太多时间， 并且毫无成就感，加上当时还在备考TOEFL，那种对时间流逝的无力感让我焦虑万分。每天回家时的疲惫感已经让我毫无斗志了，TOEFL 就这样成了一个大烂坑。8月初离职，每天跑步让焦虑缓解了不少。八月中找到一份比较轻松没什么挑战性的工作想调整一下状态，这期间跑步一直坚持了3个月，推翻了自己一直认为不适合长跑的错误观念。新到岗后还尝试过捡起 TOEFL 最后还是放弃，主要原因是新岗位的适应期的工作压力问题，但此时已经没有之前那么焦虑了。下半年读了很多书，慢慢的心态开始平复了，读到很多书都有豁然开朗的感觉，我想我的春天总算到了。

 

2.你对自己目前的处境的评价是什么样子的？是否有什么困扰或者烦心事？对于你手里在做的事情以及你每天接触到的人，这些让你的内心感受是什么样的？是一种身心愉悦的体验还是一种希望想摆脱的煎熬？或者有其他什么感受或者评价？你一天中的情绪与状态是什么样子的？主要是平稳的还是波动的？是在好的情绪时间多还是差的情绪时间多？它们有什么变化的规律？什么时候你最开心？什么时候你最不爽？

 

打破了恶性循环，步入了正常生长轨道。被财务状况所困。现在接触到都是些最底层的人，我也是其中一员。为糊口所迫有各种各样的原因，但我明白了一件事：学习是可以改变命运的。更多时候我只能是叹气，感叹太多的人放弃了挣扎，不愿意学习，只愿呆在自已的舒适区。我努力挣扎，努力学习目的就是为了摆脱目前的困境。现在我每天情绪大多时候算是平稳，在学习中获得新知识新概念时会很开心，和正在交往的女生聊天时也会很开心，发现自己做了蠢事难免会闷闷不乐，但稍加反思又可从中获得经验也就又似然了。最不爽？没网络可用，必须去网吧吸二手烟。

 

3.你当前短期内想达到的目标或者想做的事情有什么？你在三个月内有想做到的事情么，是什么？在2016年有想做到的事情么？你在更长远的一段时间里，或者十年后，有没有想过要做什么事情？把它们全部列下来，先不要想但是，不要想一个愿景后，再想一个“但是”把它掐灭。如果你是成长会2015成员，请评估一下去年写这个题目时所说的想法与完成度。

 

短期内想完成Critical thinking 和 Mindset 的阅读;

三个月内完成 iOS编程的入门工作；

2016年换行，做 iOS开发；

十年后开自己的软件公司。

 

4.在过去的一年甚至更长的时间，在成长方面，你有没有出现过行动上言行不一致的情况？或者出现过虎头蛇尾以至彻底忘记的情况？具体表现在，比如你说要做某件事情，最后不了了之；或者你挖过一些坑，然后一直没填上，然后采用再挖一个新坑的办法，填上原来的坑。对这些坑或者事情，你是怎么评价与合理化的，请你观察自己的过程。写下你挖过的烂坑，以及你是如何为这些坑找一些合理化的方法。在写的时候你不需要自责或者辩解，而是把你之前的心路变化写下来，因为可能的确有一些事情，是你战略上的需要，你才做了转移。

 

有，之前一直在做TOEFL考试准备工作，打算进新东方之类的留学培训机构。但是在准备的过程中发现自己底子差的不行（不是某一方面差，是浑身都是渣），短期内英语水平不可能有较大提升就切换到了自由生长模式。后来得知编程这种手艺不仅可以时常用英语而且可以找份收入不错的工作，于是又给自己又挖了 iOS开发的坑，打算把英语当辅助工具而不是当作饭碗。

 

5.在过去一年甚至更长时间，你有没有过什么偏见，这些偏见曾经对你有一些影响，最后由于你的行动，让你走出来，修正了偏见，然后打开了新的一扇门？请想一下，这些偏见的产生是由于什么样的原因造成的？你是如何通过行动触发了对偏见的修正？比如，你可能以为必须要在国外或者有个外教才可能学好英语，后面你在C群练了几个月朗读，发现你逐渐上路了，然后你感觉语言的环境的本质不是像一个外教那样表面。当然你修正以后的看法未必就一定是真理，但是本题目强调的是修正偏见这个过程。

 

我很长一段时间都以为思考这事情是靠自己简单绞尽脑汁就可以弄明白的事情（或者弄不明白干脆就别去想了），相当的自以为是（被第一任女友骂过这点，当时完全不明白错在哪里），完全没有正确思考的意识，更不谈什么逻辑方法了。后来接触到李笑来，知道了 critical thinking 的概念，接着读了好多书才发现自己识多么的无知，真的是脑洞大开。

 

6.我们在这个社会生存的基本模式是价值交换。我们生产产品服务，其他人购买。不管是在公司还是独立经营，都是这样一个模式。于是问题来了，如果除去你在工作上干活拿工资的事情（学生未来整体上也是遵循此模式），你认为你能输出什么样的价值？如果要选择一项技能，你用这项技能为其他人提供服务，让其他人为你付费，让市场认可你，你觉得可以是什么技能？你觉得这个技能，离你真正拿的出手并产生成交的差距还有多远？如果要弥补这个差距，你觉得自己需要做哪些事情？

剥离工作后我所剩能传递的价值：英语，iOS，驾驶。

选择 iOS 编程技能，离能找到工作还差一年时间。要在一年之内习得能独立制作 App技能：熟悉 iOS应用开发流程，了解 C 语言基础，熟悉 OC 和 Swift。

 

  

7.假使我们每个人都是独立生活在社会上，没有依赖或者依靠。那面向未来思考，你认为在未来一至三年内，你可能需要哪些其他人提供的价值或者帮助？比如在未来一年内你想在口译方面通过市面上的一些考试，那你需要一些口译方面经验丰富的人给你提供行动参询的价值；在未来一年内需要求职或者换工作，你需要关注面试简历方面的价值；在未来一年你可能有置业的需求，那你现在可能需要关注不动产投资领域；未来两年你的孩子需要入学，那你可能要开始关注学校教育等问题。请以一年内为起点，三年为上限，不要写的太远。

 

未来一年内我要转行做 iOS开发，需要一些 iOS编程经验丰富的人提供指导。同时在后续找工作期间要准备简历和筛选公司，需要相应职业指导。第二年工作后随着技术的发展可能会更深入到算法和构架层面，此时需要技术类书籍指导；第三年可以考虑结婚，这时会需要择偶方面的知识的指导。

 

8.请审视自己每天的作息规律，以及每个时间段(slot)在做什么事情，把最常见的一天你花多少时间做什么写下来。请评估这些事情的重要性，对你成长的作用。评估的意思不是问责，所以如果你一天看3小时的美剧也不用自责隐瞒。完成前面步骤后，请思考这个问题：你想要得到一些东西，比如更强的技能或者更好的生活，你要拿一些东西来换，在上述时间段(slot)中，你愿意拿哪个来换？如果要换的东西越来越多，你会先换什么，后换什么？请排出一个优先级。比如假使我要每天拿出1小时做一些其他事情，我就从原来的时间分配中砍掉比如看美剧；当我每天要拿出3小时做事情，那砍掉美剧后，再砍掉刷微博的时间……类似的。请思考，如果分别需要从你的日程中，每天腾出1、2、3、4、5小时来，你分别需要做什么样的事情？请逐一分析需要在你生活中砍掉什么东西？

 

1月14日时间统计详情：

日常活动（吃喝拉撒睡）共计：9h2m；

工作时间：10h36m;

学习时间：1h33m;

运动：1h52m;

工作占比量排第一，日常活动不能盲目减少（特别是睡眠），工作时间按60%的学习利用率来算的话，学习时间还可以加上10.6 * 0.6 = 6.36（工作的特殊性，有大把空闲时间），所以总体的学习时间为 6.36 + 1.55 = 7.91。按我现在的目标（完成 iOS入门工作）和一万小时理论来算，我要3.46年就可以从菜鸟蜕变成大神了。这是读完一个本科教育花费时间。现阶段的目标：a.完场 iOS入门，尽快找到相应工作；b.适当的运动，身体是本钱；c.搞定女朋友。要在短期内突破 iOS必须投入全部学习时间，半年内上手不会有问题。运动时间充足。女朋友怎么办？——把日常刷微博（1小时）时间抠出来，多陪准女友聊天。完毕。

 

 

9.你觉得自己的一些优点，或者比较牛逼的地方是哪些方面？有什么事例么，让你得意或者满意。或者你有什么事情是自己做过的，并且在成长方面，曾经给过自己很大的帮助。你经历过的自认为比较失败的事情有什么？可以举例。是否有什么深刻的教训，让你发现自己的不足或者让自己增长了见识？或者你觉得自己的所谓的“缺点”在哪里。注意我加了所谓的，也就是世俗意义上会认为所以你也认为是缺点的缺点了。

 

我觉得自己比较牛逼的地方是当学了一个新东西后可以马上在生活中用起来，立马就找到 high 点。譬如说知道双拼输入法比全拼输入更快后，马上选了一种，熟悉键位，用上了半天时间就完全拥抱双拼了。再例如学习美语音标也是用一天时间猛攻然后再去用。

比较失败的事情估计是两段感情，每次有点争吵或者到关键时刻我都退缩了，不想去解决问题，想到的是逃避和分手。这让我发现了性格中的一大弱点，也是现在挣扎想提高的地方，最近看了 Mindset才猛然醒悟自己是 fixed-mindset，一下子就好像找到了病根一样狂喜不已。

世俗意义上的“缺点”：不太会讲话，不够圆滑。

 

10.请描述一次你的拖延，最好是印象深刻比较经典案例风格的拖延。这里的拖延是指你最后还是把工作或者任务完成了的情况，如果仍然没有做的，请参考问题4，不算本题讨论的情况。请描写在任务初期、中期、后期以及最后完成后，你的心理状态变化以及行为表现，如果有相关的代价教训，或者哪怕是意外的收益，请都写下来。

 

最近的一次拖延恐怕就是完成这个问卷了。18日晚上就整理好了，但看着问题发了好长时间的呆：完全没法下手！到现在19日下午1点，勉勉强强完成了六个问题，每次回答问题时才发现自己的语言组织能力有多差，然而在硬着头皮写下去后却发现把东西写下来是可以让思路变得更清晰的。虽然早听过这个道理，但只有自己体验了才能真真体会到它的好处。码字不仅锻炼了语言组织能力，更是发现自己思维漏洞的一个绝好方法！！！看来作个“能说会写的持续行动者”是势在必然了。

思维模式


最近正在看一本书： Mindset —The new psychology of success,这本书围绕着 fixed-mind 与 growth-mind两个概念来讲，这两种思维不是完全孤立的，有的人在某些方面是 growth-mind，另一些方面则是 fixed-mind。Fixed-mind 会让人认为能力这件事是天生的，会就是会，不会就是不会，不要做无所谓的挣扎。可事实上呢？披头士乐队在成名之前至少积累了超过一万小时的演出经验（见马尔科姆《异类》）。同样的还有莫扎特，他成名之前有超过十年的艰苦练习。比尔·盖茨创建微软之前已经积累了相当丰富的编程经验，而爱迪生发明灯泡实验的例子已经被用的烂大街了。这些有非凡成就的人光环过于耀眼，乃至于他们的努力与辛苦都被视而不见了，心理学上还有个术语就叫光环效应。Fixed-mind 还一个非常典型的特点就是太在意外界的评价。失败会让他们觉得是对智商的否定，那肯定丢死人了，被别人贴上“傻x”的标签肯定是不能接受的，所以失败后就不会再有尝试。然后就开始装，就像小时候成绩差的反倒嘲笑成绩好的一样去贬低那些考试和参加考试的人。

反之 growth-mind 会让人认为能力可习得，连 growth-mind 本身也是可以习得的。有这种思维的人不会认为一个考试就可以定他们的终生，因为考试只是对当下能力的水平的测试罢了，只要肯花功夫准备总是会提高的。两种思维的并存看上去有点矛盾，其实这也正是挣扎的目的：理清概念间的关系，最后你会发现竟然可以用一种观念说服自己干掉另一种错误观念。

1.2 进制转换

    1.2.1 十进制转换成二进制

    将9转换成二进制

       方法：将9除以2直到商为0为止，然后将每次的余数从下往上取即可得到二进制形式。

          9 / 2 = 4   remainder 1

          4 / 2 = 2   remainder 0

          2 / 2 = 1   remainder 0

          1/ 2  = 0   remainder 1 


 余数从下往上取即可得到9的二进制 1001

     

    1.2.2 二进制转换成十进制

    求 10011011的10进制数

    方法：从低位数开始乘以2的 n-1次幂（幂数从0开始），然后相加，得到的总和即是10进制形式。

    1*2(0) + 1* 2(1) + 0 * 2(2) + 1 * 2(3) + 1 * 2(4) + 0 * 2(5) + 0 * 2(6) + 1 * 2(7) = 1 + 2 + 0 + 8 + 16  + 0  + 0 + 128 = 155


#include <stdio.h>

  printf("%i\n", number); // 155

    return 0;

}
