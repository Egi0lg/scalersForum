1.你过去的2015是什么样子的？请用三个关键词来描述一下。围绕每个关键词你可以讲一两个故事，或者对你自己的表现进行评价，包括你的行动状态，心理情绪，身体状态等各方面的稳定性以及完成了什么样的事情。

 

纠结 2015年初的时候，我产生了换工作的想法。但是由于领导的挽留和对未来的不确定，一直迟迟没有行动。但对于本职工作又依然是不满状态，纠纠结结过了一整年，对自己和工作都产生了不好的影响。

舍弃 第二个关键词跟第一个关键词其实是有关联的。因为觉得工作中太多纠结，占用了很多精力。于是在生活中舍弃了很多。特别是跟他人的链接，退回到自己的角落，纠结思考。

放纵 放纵体重暴涨，花钱不受控。写到这里，天啊，我的2015怎么如此昏暗呀。从第一个关键词的负面影响，一直到第三个关键词。所以2016年初，果断利索正式提出辞职，春节后离开。可能不是最好的决定，但负面的状态必须要改变。

 

2.你对自己目前的处境的评价是什么样子的？是否有什么困扰或者烦心事？对于你手里在做的事情以及你每天接触到的人，这些让你的内心感受是什么样的？是一种身心愉悦的体验还是一种希望想摆脱的煎熬？或者有其他什么感受或者评价？你一天中的情绪与状态是什么样子的？主要是平稳的还是波动的？是在好的情绪时间多还是差的情绪时间多？它们有什么变化的规律？什么时候你最开心？什么时候你最不爽？

 

目前的处境就是处在危险中却不自知。这个感觉主要是来自于周围的人给我的感觉。比如27岁时，却还是单身，而且由于体重很高，学历，背景，个人条件普通，情商普通，是个很不好的状态。但我自己却得过且过，没有意识到这个非常不好的处境。目前最困扰的是对于钱的渴望和自己能力之间的巨大差距。由于上以所提到的处境问题，我急于向父母证明，即使不结婚，我也可以养活自己活得不差。可能是这样急切的心理，让我对时间并没有耐心，急于求成，没有持续的努力，挖个坑换个地，老是想赚快钱，反而浪费了很多时间和精力。

目前，我身边每天接触到的人是我的同事和父母。对于父母，主要是抱着急于证明自己的态度，所以经常焦虑，觉得他们说有每一句话做的每一件事都是在不看好我，因此，常对他们态度不好。另一方面，看到他们对我20几年的爱护，又非常感动不舍，对他们态度不好后，转身就自己讨厌自己。对于我的同事，我觉得大家似乎没有像我这样的焦虑，跟他们交往时一方面羡慕他们的轻松自在，另一方面觉得无法深入沟通，没有共同语言，交流只浮在表面上。

我的情绪波动比较大，差的情绪多过好的情绪，主要会随着工作的难易程度而发生变化。解决了比较麻烦的问题时，下班后看一起喜欢的书学习一些新东西时最开心，同事不配合我的工作时最不爽。

 

3.你当前短期内想达到的目标或者想做的事情有什么？你在三个月内有想做到的事情么，是什么？在2016年有想做到的事情么？你在更长远的一段时间里，或者十年后，有没有想过要做什么事情？把它们全部列下来，先不要想但是，不要想一个愿景后，再想一个“但是”把它掐灭。

 

短期目标：1. 减肥； 2. 收入达到预计的标准；

三个月内最想做的是：1. 减肥； 2. 跑完成都半马；3. 适应新工作的节奏。

2016最想做的：1. 英语流利说； 2. 法语简单沟通无障碍； 3. 在新工作中找到感觉，适应角色； 4. 减肥成功，改变形象

更长远的时间中，我想做的事；

1.  保持健康，精力充沛的状态

2.  成为行业中的专业人士

3.  建立一个有丈夫，有小孩，有小狗，有猫咪，有院子的家庭

4.  创立一家公司，关注居家美学，如家庭关系，房屋居所，室内软装等

5.  出一本书，关于如何从混沌走到清醒，掌控自己的人生

 

4.在过去的一年甚至更长的时间，在成长方面，你有没有出现过行动上言行不一致的情况？或者出现过虎头蛇尾以至彻底忘记的情况？具体表现在，比如你说要做某件事情，最后不了了之；或者你挖过一些坑，然后一直没填上，然后采用再挖一个新坑的办法，填上原来的坑。对这些坑或者事情，你是怎么评价与合理化的，请你观察自己的过程。写下你挖过的烂坑，以及你是如何为这些坑找一些合理化的方法。在写的时候你不需要自责或者辩解，而是把你之前的心路变化写下来，因为可能的确有一些事情，是你战略上的需要，你才做了转移。

 

减肥对于减肥这件事情，从2010年一直喊到现在，曾经在2013年靠着过激的方法成功过一次，但是在1年后又打回原型。那一次成功，是因为当时的生活很简单，那一次就是为了减肥全心全意，没有想其他的事情，所以很快到达效果。但在这之后，随着工作和生活的压力，我一直对自己说，没有足够的动力去减肥了，而且有很多精力要被分散到其他地方，不能再像以前那样专心一致了。

但是，其实目前，减肥的必要性才更大，我的身体因为它已经出现了不健康的状态。

如果能真正做到，埋掉这个坑？我的合理化方法如下：1. 确立明确的合理的目标，1个月瘦20斤这种根本不合理的目标不可能完成任务。2. 分配出固定的时间，每天30分钟的运动时间挤挤还是有的，哪怕真的忙到累，那么早中晚各抽10分钟完全是可以做到的。3. 方法具体，提前计划，比如这一周7天每天分别做什么运动，把视频提前下载好，动作提前准备好，有空就动起来。4. 改变生活习惯，油，辣，糖，淀粉的摄入需要减少，少坐电梯多直楼梯，躺在床上看电脑，周末赖床，这些不好的习惯要戒掉。5. 在外面寻找其他的乐趣，以减少因无聊而引起的放肆吃喝的习惯。

专业化曾经我刚入保险行业时，告诉自己我要专业，列过很多提高专业的方法和需要学习补足的方面。但一个也没做到。懒惰，惧怕困难，专注力不高，大家告诉我主要靠关系，以及其实并没有认真去了解通过什么书籍资料去学习。

如果能真正做到，埋掉这个坑？我的合理化方法如下：1. 首先去列出需要补足的内容的框架； 2. 仔细了解什么书籍或课堂去填补这些框架； 3. 按版块，由简入难，一步步学习； 4. 重视人际关系，刻意维护会虚假，唯有真诚发心的交往，才能坚持。

 

5.在过去一年甚至更长时间，你有没有过什么偏见，这些偏见曾经对你有一些影响，最后由于你的行动，让你走出来，修正了偏见，然后打开了新的一扇门？请想一下，这些偏见的产生是由于什么样的原因造成的？你是如何通过行动触发了对偏见的修正？

 

曾经在工作中我以为，要把服务尽量做好才是应该的。但是后来有一位前辈说，如果没有投诉，意味着可能是我们服务过度，而且，服务是跟成本成正比的。不是所有的客户都需要求100%完美的服务。我对此依然不是很认同，但我也意识到力求完美只能努力，如果达不到也不必抱着天要塌下来的想法。

 

6.我们在这个社会生存的基本模式是价值交换。我们生产产品服务，其他人购买。不管是在公司还是独立经营，都是这样一个模式。于是问题来了，如果除去你在工作上干活拿工资的事情（学生未来整体上也是遵循此模式），你认为你能输出什么样的价值？如果要选择一项技能，你用这项技能为其他人提供服务，让其他人为你付费，让市场认可你，你觉得可以是什么技能？你觉得这个技能，离你真正拿的出手并产生成交的差距还有多远？如果要弥补这个差距，你觉得自己需要做哪些事情？

 

咨询服务（家庭理财方面）我的差距还很远； 需要做的事情：

1. 累积自己的财富，实践； 2. 关注民生； 3. 富有同理心的沟通；4. 专业技能提升； 5. 输出，让别人看到我；

 

7.假使我们每个人都是独立生活在社会上，没有依赖或者依靠。那面向未来思考，你认为在未来一至三年内，你可能需要哪些其他人提供的价值或者帮助？比如在未来一年内你想在口译方面通过市面上的一些考试，那你需要一些口译方面经验丰富的人给你提供行动参询的价值；在未来一年内需要求职或者换工作，你需要关注面试简历方面的价值；在未来一年你可能有置业的需求，那你现在可能需要关注不动产投资领域；未来两年你的孩子需要入学，那你可能要开始关注学校教育等问题。请以一年内为起点，三年为上限，不要写的太远。

 

不动产投资领域，互联网金融领域，民生热点，投资理财规划

 

8.请审视自己每天的作息规律，以及每个时间段(slot)在做什么事情，把最常见的一天你花多少时间做什么写下来。请评估这些事情的重要性，对你成长的作用。评估的意思不是问责，所以如果你一天看3小时的美剧也不用自责隐瞒。完成前面步骤后，请思考这个问题：你想要得到一些东西，比如更强的技能或者更好的生活，你要拿一些东西来换，在上述时间段(slot)中，你愿意拿哪个来换？如果要换的东西越来越多，你会先换什么，后换什么？请排出一个优先级。比如假使我要每天拿出1小时做一些其他事情，我就从原来的时间分配中砍掉比如看美剧；当我每天要拿出3小时做事情，那砍掉美剧后，再砍掉刷微博的时间……类似的。请思考，如果分别需要从你的日程中，每天腾出1、2、3、4、5小时来，你分别需要做什么样的事情？请逐一分析需要在你生活中砍掉什么东西？

 

早上6点闹钟响，按下闹钟，找开电脑躺在床上随便看看，7点起床，吃饭去上班； 早上，8:30-11:30 工作一小时，刷手机40分钟，去洗手间30分钟，跟自己生闷气30分钟； 午休，吃饭1小时，刷手机1小时； 晚上，不加班的时候，逛街2小时，洗澡洗头发1个半小时，边刷手机边看电视2`3小时，睡觉。

 

每天腾出2小时提高专业技能——减少刷手机时间和洗澡洗脸时间

每天腾出半小时关注民生新闻——上下班路上刷手机

每天腾出半小时运动——减少吃午饭时间

每天腾出1小时学习语言——不赖床

 

9.你觉得自己的一些优点，或者比较牛逼的地方是哪些方面？有什么事例么，让你得意或者满意。或者你有什么事情是自己做过的，并且在成长方面，曾给过自己很大的帮助。你经历过的自认为比较失败的事情有什么？可以举例。是否有什么深刻的教训，让你发现自己的不足或者让自己增长了见识？或者你觉得自己的所谓的“缺点”在哪里。注意我加了所谓的，也就是世俗意义上会认为所以你也认为是缺点的缺点了。

 

我是一个典型的包子性格，不善于拒绝，遇到问题总想自己解决不想麻烦别人，希望自己对别人是有用的。这是个缺点，但是换个角度，我因此有了一些比他人更好的解决问题的能力，我可以把这些能力加强提高后，变成可以出售的产品，这样，即在提供帮助的同时，把可以验证自己的价值，也不会因为花费时间做了很多无任何回报的事情而怪自己无能。

 

10.请描述一次你的拖延，最好是印象深刻比较经典案例风格的拖延。这里的拖延是指你最后还是把工作或者任务完成了的情况，如果仍然没有做的，请参考问题4，不算本题讨论的情况。请描写在任务初期、中期、后期以及最后完成后，你的心理状态变化以及行为表现，如果有相关的代价教训，或者哪怕是意外的收益，请都写下来。

 

最近在整理一个比较复杂的名单。一开始，我是雄心勃勃地要利用周末时间去提前预警提前搭好框架以方便后来的处理。但是，我很懒，平时工作比较拖沓，只能利用周末。而周末，又想赖床，不想去公司；工作日，也不想加班，总觉得应该来得及； 最后，一直到工作逼到脚下才开始做，可想而知，效率很差，效果不好。最后花2天熬夜把本需要1周的工作勉强做完。

我的缺点；1. 懒；  2.讨厌琐碎的事务性的工作； 3. 言行不一；4. 做事前没有想好步骤和节奏，忽视可能存在的问题，没有提前做好准备，没有从长远考虑。

2016新成员。一边参与，一边提建议吧。
